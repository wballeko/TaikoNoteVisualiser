How to use?

Character Note
d Don
k Katsu
D Don Finisher
K Katsu Finisher
r or space Rest
Use these around the notes to change them
Modifiers Effect
{ ... } Doubles the duration of the note 1/4 > 1/2
( ... ) Multiplies duration of the note by 1.5 1/4 > 1/3
[ ... ] Halves the duration of the note 1/4 > 1/8
< ... > Adds to the duration of the note 1/4 > 1/5

Note: use a combination of these to get the desired note duration
